  # ⌨ 프로젝트 소개

  ```
  "아.. 키보드 하나 때리고(?) 싶은데 종류도 너무 많고 리뷰 위주로 보고싶은데... 🤔"
  ```

  <div align="center">
  세상이 발전해가면서 it기기의 종류가 다양해지고 있습니다. <br/>
  하지만 비교적 비싼 가격 때문에 구입을 망설이며 이곳 저곳 리뷰를 찾아보는 일도 많아지고 있습니다. <br/> 이런분들을 위해 도움을 드리고자 전자기기 리뷰 서비스를 기획하게 되었습니다.
  </div>
